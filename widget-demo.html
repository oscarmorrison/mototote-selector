MotoTote Widget Demo

How to embed this widget in your Shopify website:

  1. Build the widget: yarn build:widget
  2. Upload the generated dist/mototote-widget.js file to your Shopify theme assets
  3. Choose one of the methods below to add the widget

Method 1: Custom Element (Recommended)

Add this script tag to your theme's theme.liquid or page template:

<script src="{{ 'mototote-widget.js' | asset_url }}"></script>

Then add this HTML where you want the widget:

<mototote-widget></mototote-widget>

Method 2: Fallback Container (If CSP blocks custom elements)

Add this script tag to your theme's theme.liquid or page template:

<script src="{{ 'mototote-widget.js' | asset_url }}"></script>

Then add this HTML where you want the widget:

<div id="mototote-widget-container"></div>

Fallback Container Demo:

Method 3: Manual Initialization

Add this script tag to your theme's theme.liquid or page template:

<script src="{{ 'mototote-widget.js' | asset_url }}"></script>

Then add this HTML where you want the widget:

<div id="my-widget-container"></div>

And call this JavaScript:

initMotoToteWidget('my-widget-container');

Manual Initialization Demo:

For Shopify JSON Templates:

Add this to your page template JSON:

{
  "sections": {
    "mototote_widget": {
      "type": "custom-liquid",
      "settings": {
        "custom_liquid": "<script src=\"{{ 'mototote-widget.js' | asset_url }}\"></script><div id=\"mototote-widget-container\"></div>"
      }
    }
  },
  "order": ["mototote_widget"]
}
